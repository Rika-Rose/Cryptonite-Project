# Cryptonite: Real-Time Crypto Analytics Dashboard
A Single-Page Application (SPA) designed for tracking, filtering, and visualizing live cryptocurrency market data.

Technical Implementation:
Dynamic Data Integration: Orchestrated asynchronous REST API calls to fetch live market data, coin specifics, and historical price points.

Interactive State Logic: 
Engineered a custom selection system allowing users to track up to 5 specific coins, featuring real-time validation and UI synchronization.

Real-Time Visualization: 
Leveraged data-mapping techniques to generate live reporting charts for currency comparison.

Modular Architecture: Organized code into specialized utility modules to ensure maintainability and separation of concerns.
